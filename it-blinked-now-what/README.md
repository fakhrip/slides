# It Blinked. Now What?

Slidev deck for explaining how embedded ideas move from PoC to MVP, PMF, sales, and post-revenue execution.

## Run locally

```bash
npm install
npm run dev
```

## Export

```bash
npm run export
```

The deck is in `slides.md`. Images are in `public/` so Slidev frontmatter/background paths work after build.
